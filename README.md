# tcdr
"TsConfig.json Don't Read" -- A cli to check the usage of tsconfig.json built with [fx](https://github.com/antonmedv/fx).

> 单纯自己试手做的一个小命令行工具，提升自己的 `typescript` 开发体验。

## 用法
```sh
# 展示所有的 tsconfig.json 的选项
tcdr all

# 展示特定的选项具体定义，如 files
tcdr files
```
